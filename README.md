# 🧰 code-for-simulation

> Personal code library by Woojin Jung for quick experimentation and reuse

## 📌 What is this repository?

이 저장소는  
👉 **연구 및 실험 과정에서 필요할 때 꺼내 쓰기 위해 만든 코드 모음집**입니다.

특정 주제에 한정된 레포가 아니라,

- 이미지 처리
- FFT / 주파수 분석
- Grad-CAM
- filtering
- simulation
- 기타 아이디어 실험 코드

처럼  
👉 **연구하면서 그때그때 만든 범용 코드들을 모아둔 개인 라이브러리**입니다.

---

## 🧠 Why this exists

연구하다 보면 계속 반복되는 상황:

- 예전에 짰던 코드 다시 필요함
- 간단한 테스트인데 새로 구현하기 귀찮음
- 아이디어 검증을 빠르게 해보고 싶음

그래서 만든:

> 👉 "연구자를 위한 코드 스니펫 저장소"

---

## 📂 Characteristics

- 특정 프로젝트 ❌  
- 일관된 구조 ❌
